Что выполнено по боту

Ubuntu

На странице https://ubuntu.com/download/desktop взята версия Ubuntu 24.04.4 LTS.

В Virtualbox создана виртуальная машина с именем Ubuntu 24 и характеристиками:

  • 8 Гб оперативы;
  • 25 Гб жесткий диск;
  • 2 процессора.

К виртуальной машине подключен образ Ubuntu 24.04.4 LTS.

Rust

Установлен Rustup:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Установлена цепочка инструментов Rust:

rustup toolchain install stable

Выполнена проверка установки инструментов:

cargo --version cargo 1.92.0 rustc --version rustc 1.92.0

Visual Studio Code

Для установки Visual Studio Code выполнены команды:

sudo apt-get install wget gpg &&
wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg &&
sudo install -D -o root -g root -m 644 microsoft.gpg /usr/share/keyrings/microsoft.gpg &&
rm -f microsoft.gpg

Создан файл /etc/apt/sources.list.d/vscode.sources следующего содержания:

Types: deb
URIs: https://packages.microsoft.com/repos/code
Suites: stable
Components: main
Architectures: amd64,arm64,armhf
Signed-By: /usr/share/keyrings/microsoft.gpg

Выполнены команды:

sudo apt install apt-transport-https
sudo apt update
sudo apt install code

База данных

Базовая установка

Установка базы данных PostgreSQL на виртуальной машине:

sudo apt update
sudo apt install postgresql

Настройка TCP/IP

Выполнена настройка прослушивания по протоколу TCP/IP вместо сокетов...

sudo nano /etc/postgresql/<версия>/main/postgresql.conf

...и в открывшемся файле раскомментирована строка:

listen_addresses = 'localhost'

...и закомментирована строка:

unix_socket_directories = '/var/run/postgresql'

Установка SQLX

В Cargo.toml добавлена ссылка на библиотеку для работы с базами данных...

sqlx = { version = "<последняя версия>", features = ["postgres"] }

...и установлен консольный инструмент для этой библиотеки:

cargo install sqlx-cli --no-default-features --features postgres

Создание миграции

Перешли в папку проекта:

cd ~/Repository/Rust/kartoteka-bot

В папке проекта создана миграция для таблицы ботов...

cargo sqlx migrate add create_bots_table

...и заполнено её содержимое:

CREATE TABLE IF NOT EXISTS bots (
id SERIAL PRIMARY KEY,
name CHAR(50) UNIQUE NOT NULL,
token VARCHAR(100) NOT NULL
);

Файл переменных окружения

Создан файл переменных окружения .env:

touch ~/Repository/Rust/kartoteka-bot/.env

Файл прописан в .gitignore, чтобы конфиденциальные данные об окружении не утекли в систему контроля версий: Содержимое .gitignore:

/target
.env

Пароль

Задан пароль для доступа к базе данных:

sudo -u postgres psql

ALTER USER postgres WITH PASSWORD '<пароль>'; \q

База данных

Создана база данных:

sudo -u postgres psql

CREATE DATABASE <имя БД>; \q

Итоговая строка

В .env файле прописана итоговая строка подключения к базе данных:

DATABASE_URL=postgres://postgres:<пароль>@localhost:5432/<имя БД>

Настройка работы с базой данных в программе

Создан файл database.rs, описан пул подключений к базе данных. Создан файл model.rs, описаны модели данных в базе данных. Создан файл handlers.rs, определены обработчики сетевых запросов к базе данных.

В основном файле main.rs добавлены объявления данных модулей и настроено получение строки подключения из переменных окружения или файла переменных окружения, а также реализовано создание пула подключений к базе данных.
